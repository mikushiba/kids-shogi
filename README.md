# こどもしょうぎ

## これは何か
小学校低学年向けの本将棋（9×9）ゲーム。ブラウザだけで動く1ファイル（index.html）のWebアプリで、インストール不要。ルールを覚えていなくても、駒をタッチすると進める場所が光るので遊びながら覚えられる。

## 誰向けでいつ使うか
- 小学校低学年の子供が将棋で遊ぶとき（ふりがな付き・ひらがな表記）
- はじめてでルールを覚える段階＝れんしゅうモード（駒を減らした4ステージ。歩だけ→飛角→金銀→桂香の順に1種類ずつ増える。勝つと次のステージへ進める）
- 1人で遊ぶとき＝コンピュータ対戦（子供が勝てる弱さに調整済み）
- 親子・兄弟で遊ぶとき＝ふたりで対戦モード
- 詰みの形を覚える練習＝つめしょうぎ100問（1手詰め70問＋3手詰め30問。クリアした問題は緑の番号で記録される）

## 使い方
- 公開URL（スマホ・タブレット・PCのブラウザで開くだけ）: https://mikushiba.github.io/kids-shogi/
- ローカルで開く場合: この `index.html` をダブルクリックしてブラウザで開く

### 画面の機能
| ボタン | はたらき |
|---|---|
| あそびかた | 子供向けのルール説明 |
| こまのうごき | 全部の駒の動き方ずかん（成り駒も） |
| 🔊 | 効果音のオン・オフ |
| まった | 1手前にもどる（CPU戦では自分の手番までもどる） |
| はじめから | 対局をやりなおす |
| つづきから あそぶ | とちゅう保存が残っているときだけ、最初の画面に出る。前回の続きから再開する |

### つめしょうぎ（100問）
最初の画面の「🧩 つめしょうぎ 100もん」から。相手の玉を詰ませたら正解で、どの手でも本当に詰めば正解になる。間違えると自動でやりなおし。3手詰めでは自分の手はぜんぶ王手でないといけない（本物の詰め将棋のルール）。相手はいちばん粘る受けを自動で指す。クリアした問題はブラウザに記録され、一覧で緑の番号になる。「💡 こたえ」を押すと正解手順が自動で動いて見られる（見た場合はクリア扱いにならない）。

問題は市販の問題集のコピーではなく、このゲーム自身のルールエンジンで自動生成し、100問すべて「本当に詰むか」を機械検証してある。出題順は子ども向け詰め将棋の定石（頭金→銀→飛→角→桂→香→と金の順に、駒の種類ごとに詰みの形を覚える）に合わせた。生成の仕組みは `dev/tsume_gen.mjs`（`node dev/tsume_gen.mjs` で再生成できる。乱数の種を固定してあるので同じ結果が再現される）。

埋め込み済みデータの検証は、ブラウザで `index.html?test=tsume` を開くと自動で走り、「つめしょうぎ検証: 100/100 合格」と表示される。

### とちゅう保存
1手ごとに自動でブラウザ内（localStorage）へ保存する。子供が保存ボタンを押し忘れても消えない。次に開くと最初の画面に「つづきから あそぶ」が出て、盤面・持ち駒・手番・まった履歴がそのまま戻る。勝敗がついた対局の保存は消えるので「つづきから」には出ない。保存枠は1つで、新しい対局を始めると前の保存は上書きされる。保存先は端末の中だけで、外部へは送らない。

### 入っている将棋ルール
本物の将棋ルールを実装している。合法手のハイライト、持ち駒（打ち）、成り（強制成り含む）、二歩の禁止、打ち歩詰めの禁止、王手放置の禁止、詰み判定。千日手と入玉宣言は子供向けのため未実装。

## トリガーと成果物
- トリガー: ブラウザでURLを開く（自動実行や定期処理はない）
- 成果物: 画面上の対局のみ。とちゅう保存は使っている端末のブラウザ内（localStorage、キー `kodomo-shogi-save-v1`、約2KB）にだけ書く。サーバーへの通信・外部送信はゼロ

## 関連ファイル
- `index.html` … ゲーム本体（HTML/CSS/JS全部入り。正本。つめしょうぎの問題データも埋め込み済み）
- `dev/tsume_gen.mjs` … つめしょうぎ問題の生成・検証スクリプト（開発用。遊ぶだけなら不要）
- `dev/tsume_data.js` … 生成された問題データ（index.htmlへ埋め込む中間ファイル）
- GitHubリポジトリ … https://github.com/mikushiba/kids-shogi （バージョン管理の正本）
- 公開ページ … GitHub Pages（mainブランチにpushすると1〜2分で自動反映）
